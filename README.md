1. DESKRIPSI APLIKASI
Aplikasi ini merupakan aplikasi pemutar musik berbasis konsol yang dibuat menggunakan bahasa pemrograman C++
Aplikasi ini menerapkan struktur data Multi Linked List dan Double Linked List

Ada dua peran disini yaitu Admin dan User
Admin bertugas untuk mengelola lagu seperti  menambahnkan lagu, mengedit nama judul/artis lagu, menghapus lagu
User dapat membuat playlist nya berdasarkan lagu lagu yang dibuat oleh admin di library lagunya dan bisa memutar lagunya dari playlist user

2. FITUR APLIKASI
1. Fitur Admin
   - Menambahkan lagu ke library
   - melihat daftar lagu
   - menghapus lagu dari library
   - mengedit lagu

2. Fitur User
   - melihat lagu di library
   - menambahkan lagu dari library ke playlist
   - membuat dan melihat isi playlist
   - memutar lagu dari playlist

  - buka codeblocks
